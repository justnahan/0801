# 產品需求文檔 (Product Requirements Document) - 夢境市集

> 💡 **給 Vibe Coder 的說明**: 歡迎來到夢境市集的設計空間！這將是一個超現實的購物體驗，讓用戶仿佛漫步在夢境中。

## 🎯 核心設計理念

夢境市集是一個突破傳統購物體驗的超現實電商平台，將購物轉化為一場奇幻的夢境之旅。目標受眾是喜歡獨特體驗、追求創意生活的年輕消費者，特別是對藝術、設計、手工藝品有興趣的用戶。整個網站要營造出飄逸、夢幻、療癒的氛圍，讓購物變成一種享受和探索。

## 🏗️ 頁面佈局與結構

### 導航區域
頂部導航採用星空中的天體作為導航元素：
- 月亮圓形按鈕作為主要 Logo，帶有柔和的銀白色光暈
- 主要導航項目設計成星座形狀的按鈕組（首頁、商品、關於我們、購物車）
- 搜索框設計成望遠鏡的形狀，內部有星塵粒子效果
- 用戶頭像區域是小行星帶的概念，懸浮在右上角

### 主要內容區域
- 背景是深邃的漸變星空（深紫 → 靛藍 → 深綠）
- 極光效果緩慢流動穿越整個背景
- 商品網格佈局，每個商品都「坐」在半透明的雲朵上
- 雲朵會輕柔地上下漂浮（CSS 動畫循環 3-5 秒）
- 中央區域預留「心情探測器」功能區

### 頁尾區域
- 設計成夢境的地平線，漸變消失在遠方
- 聯絡資訊和連結以流星雨的形式散佈
- 版權資訊用星塵字體顯示
- 社交媒體按鈕設計成不同的小星球

## 🛍️ 商品展示方式

### 商品卡片設計
- 每個商品卡片都置於一朵半透明的白色雲朵上
- 商品圖片採用圓角設計，帶有輕微的發光邊框
- 商品名稱使用夢幻字體，帶有微弱的發光效果
- 價格標籤設計成小星星的形狀，金色光澤
- 卡片整體具有玻璃質感（frosted glass effect）

### 商品列表佈局
- 桌面版：響應式網格，3-4 列排列
- 商品卡片大小略有差異，營造不規則的夢幻感
- 卡片之間的間距不完全均勻，模擬雲朵的自然分佈
- 無限滾動載入，新商品從遠方的雲層中「飄」出來

### 互動效果
- 滑鼠懸停：雲朵發出柔和的光暈，商品輕微放大（1.05 倍）
- 產生彩虹色的光環圍繞商品
- 點擊時爆出星星粒子效果
- 加入購物車時，商品會「飛」向右上角的購物車圖標

## 🎨 視覺風格與品牌

### 色彩方案
- **主色調**：深紫色（#2D1B69）、靛藍色（#0F0C29）
- **極光色彩**：粉紫色（#FF6B6B）、青綠色（#4ECDC4）、薄荷綠（#A8E6CF）
- **輔助色**：星光白（#F8F8FF）、月光銀（#C0C0C0）、星塵金（#FFD700）
- **強調色**：夢幻粉（#FFB6C1）用於重要按鈕和提示

### 字體選擇
- **標題字體**：使用帶有輕微發光效果的無襯線字體，字重較輕，營造飄逸感
- **正文字體**：清晰易讀的圓體字，保持夢幻但不影響閱讀
- **特殊文字**：價格和重要信息使用帶有星塵效果的字體
- **所有文字都帶有微弱的外發光效果**

### 整體氛圍
網站要傳達寧靜、夢幻、治癒的感覺，讓用戶感受到：
- 漂浮在雲端的輕盈感
- 置身夢境的超現實體驗
- 溫暖而神秘的購物氛圍
- 藝術與商業的完美結合

## ✨ 用戶互動與動畫

### 按鈕和連結
- 主要按鈕採用透明玻璃質感，邊框發光
- 懸停時按鈕會輕微浮起，產生陰影效果
- 點擊時產生漣漪效果，配合星塵散落動畫
- 導航按鈕懸停時星座會完整顯現

### 過渡動畫
- 頁面切換採用夢境轉場效果：當前頁面溶解成星塵，新頁面從遠方凝聚而成
- 商品載入時從雲層深處飄出來
- 所有元素都有輕微的浮動動畫（不同頻率，避免統一性）
- CSS 使用 ease-in-out 曲線，模擬夢境中的緩慢節奏

### 載入效果
- 主載入動畫：旋轉的星系，中央有小星星匯聚
- 局部載入：雲朵形狀的載入指示器，內部有星塵流動
- 數據載入時顯示「正在穿越夢境...」等趣味文字

## 📱 響應式設計需求

### 手機版
- 商品改為單列瀑布流顯示
- 導航收縮為漢堡菜單，點開時展現星空菜單
- 極光背景簡化，減少動畫以節省電量
- 觸摸互動優化：點擊產生更明顯的視覺反饋

### 平板版
- 商品顯示 2-3 列網格佈局
- 保持大部分桌面端的視覺效果
- 觸摸手勢支持：雙指縮放查看商品細節
- 橫豎屏切換時佈局平滑過渡

### 桌面版
- 完整的 4 列商品展示
- 所有動畫效果全開
- 滑鼠游標特效：流星軌跡和星塵
- 鍵盤快捷鍵支持（如空格鍵暫停/播放背景動畫）

## 🎯 特殊功能需求

### 心情探測器
- 用戶進入網站時的歡迎彈窗
- 提供 5-6 種心情選項（快樂、平靜、憂鬱、興奮、思考、夢幻）
- 根據選擇調整背景極光的顏色和流動速度
- 影響商品推薦的權重算法

### 夢境音效系統
- 環境背景音：風鈴聲、遠處的音樂盒、輕柔的合成器音樂
- 互動音效：點擊星星的清脆聲、雲朵碰撞的柔和聲
- 音效開關控制，尊重用戶偏好
- 音量會根據用戶活動自動調節

### 時間感知功能
- **黃昏夢境**（18:00-22:00）：溫暖的橙粉色極光
- **午夜夢境**（22:00-06:00）：深邃的藍紫色調
- **黎明夢境**（06:00-12:00）：清新的青綠色光芒
- **午後夢境**（12:00-18:00）：明亮的彩虹色彩

### 其他功能
- 商品篩選器設計成「夢境望遠鏡」
- 購物車頁面是「夢境收藏盒」的概念
- 用戶評價系統使用星級評分，星星會真的發光
- 分享功能：生成用戶專屬的「夢境明信片」

## 💬 其他補充說明

### 開發階段規劃

**第一階段：夢境基礎架構**
- 建立基礎的星空背景和極光動畫
- 實現商品卡片的雲朵設計和基本佈局
- 完成響應式網格系統
- 基礎的懸停和點擊效果

**第二階段：交互魔法增強**
- 實現滑鼠軌跡特效
- 完善所有動畫過渡效果
- 添加粒子系統（星塵、星星爆炸等）
- 頁面轉場動畫

**第三階段：特殊功能實現**
- 心情探測器功能
- 時間感知系統
- 音效系統集成
- 高級篩選和搜索功能

**第四階段：體驗優化**
- 性能優化，確保動畫流暢
- 細節打磨和用戶體驗優化
- 跨瀏覽器兼容性測試
- 無障礙功能支持

### 技術考量
- 使用 CSS3 動畫和 WebGL 技術實現復雜視覺效果
- 考慮到動畫性能，提供「簡化模式」選項
- 音效使用 Web Audio API 實現
- 響應式設計使用 CSS Grid 和 Flexbox

---

> 📌 **提示**: 
> - 這個設計重點在於營造沉浸式的夢幻體驗
> - 所有動畫都應該是柔和、緩慢的，符合夢境節奏
> - 色彩運用要大膽但和諧，避免過於刺眼
> - 記得在每個開發階段都保持夢幻主題的一致性！
